
# Sublist3r Docker image
A Docker image of [Sublist3r](https://github.com/aboul3la/Sublist3r), for enumerating subdomains.

## Usage:

    docker run --rm papko26/sublist3r -d <domain>
    
## Example:

    ~: docker run --rm papko26/sublist3r -d example.com
